# 从尾到头打印链表

输入一个链表，按链表从尾到头的顺序返回一个ArrayList。

```python
# -*- coding:utf-8 -*-
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    # 返回从尾部到头部的列表值序列，例如[1,2,3]
    def printListFromTailToHead(self, listNode):
        # write code here
        l = []
        if not listNode:
            return l
        while listNode:
            l.append(listNode.val)
            listNode = listNode.next
        return l[::-1]
            
            
```
